# Teste_QA

## Sobre

Repositório do treinamento: Workflow de testes contínuos em Cypress de API no Github Actions

## Stacks
- Cypress
- Javascript

## Rodando

1. Clonar o repositório

2. Instalar as dependências do Node
```
npm install
```

3. Subir o Cypress UI
```
npx cypress open
```

4. Executar testes em Headless
```
npx cypress run 
```
